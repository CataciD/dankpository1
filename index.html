API training project - timestamp API

by Catacid

You are supposed to pass a string in the URL.
This dumbo is supposed to check if it's a date and if it is return a json or smth.

The URL to parse should look something like this:

https://camper-api-project-catacid.c9users.io/December%2015,%202015
https://camper-api-project-catacid.c9users.io/1450137600

The result should look like this:

{ "unix": 1450137600, "natural": "December 15, 2015" }
